### Phase 2: 后端基础

---

#### I4: SQLite 本地存储层

**目标**: 实现 SQLite 持久化层

**任务**:
1. 添加依赖:
   ```toml
   [dependencies]
   sqlx = { version = "0.7", features = ["runtime-tokio", "sqlite"] }
   ```

2. 创建存储模块结构:
   ```
   src-tauri/src/storage/
   ├── mod.rs
   ├── connection.rs    # 连接配置存储
   └── migrations.rs    # 数据库迁移
   ```

3. 实现连接配置表:
   ```sql
   CREATE TABLE IF NOT EXISTS connections (
       id TEXT PRIMARY KEY,
       name TEXT NOT NULL,
       type TEXT NOT NULL CHECK(type IN ('mysql', 'postgresql')),
       host TEXT NOT NULL,
       port INTEGER NOT NULL,
       username TEXT NOT NULL,
       password TEXT NOT NULL,  -- 加密存储
       database TEXT,
       environment TEXT CHECK(environment IN ('production', 'staging', 'development')),
       tags TEXT,  -- JSON array
       created_at TEXT NOT NULL,
       updated_at TEXT NOT NULL
   );
   ```

4. 实现 CRUD 操作:
   - `init_db()` - 初始化数据库
   - `list_connections()` - 列出所有连接
   - `get_connection(id)` - 获取单个连接
   - `create_connection(data)` - 创建连接
   - `update_connection(id, data)` - 更新连接
   - `delete_connection(id)` - 删除连接

**产出文件**:
- `src-tauri/src/storage/mod.rs`
- `src-tauri/src/storage/connection.rs`
- `src-tauri/src/storage/migrations.rs`

**验证命令**:
```bash

```

       pub port: u16,
       pub username: String,
       pub password: String,
       pub database: Option<String>,
   }

## Docker 测试环境

创建 `docker-compose.yml` 用于数据库连接测试:

```yaml
version: '3.8'
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: testpass
      MYSQL_DATABASE: testdb
    ports:
      - "3306:3306"
    healthcheck:
      test: ["CMD", "mysqladmin", "ping", "-h", "localhost"]
      interval: 5s
      timeout: 5s
      retries: 5

  postgres:
    image: postgres:14
    environment:
      POSTGRES_PASSWORD: testpass
      POSTGRES_DB: testdb
    ports:
      - "5432:5432"
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U postgres"]
      interval: 5s
      timeout: 5s
      retries: 5
```

**使用方式**:
```bash
# 启动测试数据库
docker-compose up -d

# 运行集成测试
cargo test -- --ignored

# 停止测试数据库
docker-compose down
```
